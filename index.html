Trabalho 1 - Página Portfólio

AndreMaciel66

Desenvolver uma página web utilizando a linguagem HTML utilizando as tags aprendidas em sala e outras que o aluno achar necessário. A página web deverá ser um portfólio do próprio aluno e deverá conter pelo menos 3 páginas ligadas entre si, sendo de preferência:

Home +Info

Página inicial com conteúdos mais genéricos e opção de acesso as outras opções.

Sobre mim +Info

Descrição pessoal, currículo e experiência que o profissional possui.

Portfólio +Info

Projetos já desenvolvidos pelo profissional, normalmente com uma descrição e uma linguagem do projeto, assim como um link para encaminhar o profissional para o projeto.
